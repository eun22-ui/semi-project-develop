> ### 주요기능
##### 1.   메인페이지(index)

   
       인기도서 / 도서관 소식 / 추천서평
##### 2.    로그인 / 회원가입 / 마이페이지 


       도서대출 / 도서반납 / 도서예약 / 도서신청 > 현황확인
##### 3.   자료검색

    
       통합 및 간편 검색 / 카테고리 검색 / 신규도서 / 추천도서
##### 4.   사용자 기능

    
       도서 대출 / 반납 및 예약 / 도서 신청 
##### 5.  문의 및 소통

    
       공지사항 / 문의게시판 / 한줄평(후기) / 자유게시판
##### 6.  관리자 기능

    
       도서 관리 
##### 7.  도서관 안내(정보)

     
       도서관 소개 / 오시는 길

<br>


> ### 담당역할
##### 주제선정 및 데이터 구축, 게시판 및 커뮤니티 부분을 담당했습니다.

* 공지사항 및 첨부파일 (CRUD) / 관리자 권한 제어
* 문의게시판 (CRUD) / 관리자, 회원 문의 및 답변 설정 
* 한줄평 (CRUD) 
* 자유게시판 (CRUD)
* 도서관 안내 
* 온프레미스 환경 구축 – 서버 / 데이터베이스
